# Portfoli Personal 💻

## Descripció
Pàgina web personal per mostrar els meus projectes i habilitats
com a estudiant d'ASIX.

Projecte personal realitzat per aprendre desenvolupament web
i presentar el meu treball com a estudiant.

- CSS3 (disseny responsive)
- Google Fonts

## Captures
Inclou secció hero, projectes, sobre mi i contacte.

## Com executar-lo
Descarrega l'arxiu `portfoli-personal.html` i obre'l amb qualsevol navegador.
